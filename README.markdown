# PHPUnit All in One

## How to install

	git clone git@github.com:EHER/phpunit-all-in-one.git
	cd phpunit-all-in-one
	./install.sh

or

	git clone git@github.com:EHER/phpunit-all-in-one.git
	cd phpunit-all-in-one
	git submodule init
	git submodule update


## Running phpunit 

	./phpunit.php

or

	/path/to/phpunit-all-in-one/phpunit.php
