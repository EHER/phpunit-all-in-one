# PHPUnit All in One

That's another way to install PHPUnit without the PEAR


## How to install

	git clone git@github.com:EHER/phpunit-all-in-one.git
	cd phpunit-all-in-one
	make

or

	git clone git@github.com:EHER/phpunit-all-in-one.git
	cd phpunit-all-in-one
	git submodule update --init


## Running phpunit 

	./phpunit.php

or

	/path/to/phpunit-all-in-one/phpunit.php
